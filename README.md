# Email Validator (Python)

A simple Python program that checks whether an email address is valid or not using regular expressions.


## Features
- Validates email format
- Checks for correct structure (name@domain.com)
- Beginner-friendly Python project
- Fast and simple to use
  

## Built With
- Python 3
- Regular Expressions (re module)


## How to Run

1. Clone the repository:
bash
git clone https://github.com/your-username/email-validator-python.git


2. Go to project folder:
bash
cd email-validator-python


3. Run the file:
bash
python main.py


## Example Output

Enter your email: test@gmail.com
Valid Email ✔


## Purpose
This project is created to learn Python basics and understand how email validation works using regex.
